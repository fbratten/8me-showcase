How the Orchestrator Flow Works

User Request enters the system and is routed by the Task Manager into the Pending Queue.

The Claude Executor picks up tasks and processes them. Results pass through Verification -- a key Ralph Loop principle: never trust AI self-assessment.

Verified tasks move to Completed. Failed verification routes to the Retry Queue for another attempt. The Circuit Breaker prevents infinite loops by capping retries.
