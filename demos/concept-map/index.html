Reading the Map

Nodes represent core concepts taught across 8me's 15 labs. Node size reflects how many other concepts depend on it.

REQUIRES edges mean you need to understand the source before the target. ENABLES edges show what new capabilities each concept unlocks.

Start from the foundation group (loop-pattern, verification) and follow edges outward to see the full learning path.
